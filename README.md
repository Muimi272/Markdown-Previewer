# Markdown Previewer / Markdown 预览器

English

A lightweight, offline Markdown Previewer implemented as a single HTML file. Open Markdown_Previewer.html in any modern browser to edit Markdown and see a live preview.

中文

一个轻量、可离线使用的 Markdown 预览器，基于单个 HTML 文件实现。打开 Markdown_Previewer.html 即可在任意现代浏览器中编辑 Markdown 并实时预览。

---

English — Quick Start

1. Download or clone this repository.
2. Open the file Markdown_Previewer.html in your browser (double-click or right-click → Open with).
3. Type or paste Markdown into the editor area on the left. The rendered HTML preview appears on the right.

中文 — 快速开始

1. 下载或克隆此仓库。
2. 在浏览器中打开 Markdown_Previewer.html 文件（双击或右键 → 使用…打开）。
3. 在左侧编辑区输入或粘贴 Markdown，右侧将显示渲染后的实时预览。

---

English — Features

- Single-file offline app (HTML/CSS/JS). No build step required.
- Live preview: edits update immediately.
- Supports common Markdown syntax (headings, lists, code blocks, links, images, etc.).
- Minimal, responsive UI that works on desktops and tablets.

中文 — 功能

- 单文件离线应用（HTML/CSS/JS）。无需构建步骤。
- 实时预览：编辑即时更新。
- 支持常见 Markdown 语法（标题、列表、代码块、链接、图片等）。
- 极简响应式界面，兼容桌面和部分平板设备。

English — Usage Tips

- To test local images, use relative file paths or drag images into supported areas if implemented.
- For large documents, consider pasting in sections to keep the editor responsive.
- Use fenced code blocks (```lang) for syntax-highlighted code if the previewer includes a highlighter.

中文 — 使用小提示

- 若需预览本地图片，请使用相对路径或在支持的实现中拖拽图片。
- 对于大型文档，可分段粘贴以保持编辑器响应性。
- 使用围栏代码块（```语言）以在启用语法高亮的预览器中获得更好显示效果。

English — Development & Customization

If you want to customize the app (styles, behavior, add plugins):
1. Open Markdown_Previewer.html in a text editor.
2. Modify the HTML/CSS/JS directly. Changes take effect by reloading the file in the browser.
3. Optionally extract CSS/JS into separate files and add build tooling if you prefer a multi-file project.

中文 — 开发与定制

若想定制应用（样式、功能、添加插件）：
1. 在文本编辑器中打开 Markdown_Previewer.html。
2. 直接修改 HTML/CSS/JS，保存后在浏览器中刷新页面以查看效果。
3. 若偏好多文件项目，可将 CSS/JS 分离为独立文件并添加构建工具链。
